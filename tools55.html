Tools 5.5




  1. History

    When Borland released the free command-line tools from C++ Builder 5, many people asked if they can be used to upgrade the old Borland C++ 5.x products.
    It turned out that direct replacement is not possible, because BC5 IDE uses COMP32.DLL, not the command-line compiler BCC32.EXE.
    The only way to use the new compiler tools is to register them as an external tools in BC5 IDE, and associate them with the proper nodes. To make this process easy, Karst Drenth created an Add-On for Borland C++ 5.02.

  2. Product

    This Add-On is called Tools 5.5 and can be used for free. The instalation can be downloaded here.
    To see a list of the changes, refer to the readme file.

  6. Specific information

    1. Tools 5.5 and Codeguard

      Tools 5.5 targets can use only Codeguard DLL and LIB from C++ Builder 5/6. The old Codeguard files from Borland C++ 5.x won't work.
      To use Codeguard, copy CG32.LIB from C++ Builder LIB folder to BCC 5.5 LIB folder, so it can be linked.
