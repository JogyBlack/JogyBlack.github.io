Required and recommended downloads

Free compiler

First you need the free compiler (the most recent one is Borland C++ 5.5.1). About 9 MB are awaiting you at http://www.borland.com/bcppbuilder/freecompiler/ (You are forced to become a member but that's free too)

Since the first release of the free commandline tools Borland has published 2 patches.

  • If you're downloading today no update should be necessary.
  • The very first downloads (before April 2000) have some shortcomings that can be fixed through installation of Service Pack 1. The newer downloads already contain the updated materials (improved help files, some additional headers).
  • All downloads up to Aug 24th should be patched with the brand new Service Pack 2.

Turbo Debugger

The free compiler download does not include a debugger. Borland has re-released the original Turbo Debugger for FREE for use with Borland C++ Compiler 5.5

Helpfiles

I recommend to get the complete set of help files for the Builder product from http://www.borland.com/techpubs/bcppbuilder or http://www.borland.com/devsupport/bcppbuilder/. Check this for updates and service packs too!

FAQs

I recommend to get New C/C++ and C++Builder FAQs (This can exceed 1MB of data and take a while to download. The complete list is also available as a zipped file). FAQs means Frequently Asked Questions and these FAQs is really a very useful resource for answering questions you might encounter eventually. Look up your questions in the FAQs before searching the news or posting to borland.public.cppbuilder.commandlinetools.

Editor/Environment

For writing and modifying sources an additional editor is required. The following tools integrate an editor with support for driving the compiler and linker:
  • VIDE An Integrated Development Environment (IDE) with special support for the free borland compiler. Supports other compilers/languages as well. Special Features: Syntax highlighting, code browsing (searching for definitions among different files), generates makefiles for simple projects.
  • Emacs for Win32
  • Padded A simple IDE dedicated to the free borland compiler. Features: Syntax highlighting; configurable context sensitive help
  • Mini-Ide A minimal IDE for the free borland compiler.
  • Quincy 2000 IDE for C/C++ compilers.
The bare minimum is a simple text editor like Notepad.

Libraries for interfacing with the Win32 GUI

The free compiler includes neither OWL nor VCL. If you have an older version of OWL you might be able to upgrade to OWL NExt. Nevertheless these free libraries might be useful:
  • wxWindows is "a free C++ framework to make cross-platform programming child's play". You can download an archive containing sources, samples and even makefiles.
     
  • V C++ GUI Framework - the library used to create VIDE. V is under the GNU license. The author of V - Bruce E. Wampler - describes it as
     
    [...] small, elegant [...] designed to be portable. Currently, versions for the X Windowing System (using a customized 3D Athena widget set), Microsoft Windows 3.1, and Microsoft WIN32 (Windows 95 and NT) are available. [...]

     
  • VXCL - formerly named Virtual Windows Class Library (VWCL) - is a free, portable, open source library for creating GUIs. It includes classes for frame windows, controls, dialogs and much more.
