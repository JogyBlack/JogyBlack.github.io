Practical introduction

After reading these pages you will be able to create console mode as well as GUI mode applications using the free commandline tools.

Preparations

Create an extra directory for the samples presented on this page. Make sure that Bcc32 is somewhere in your PATH. Use an editor which can save as plain text. If you chose Notepad be careful to doubleqoute filenames in the SaveAs dialog. To save your code as hello.c  you should enter "hello.c"

Get started with MAKE

With more sources the process of making a program can be tedious and error prone. Visual environments usually introduce the concept of a project which can be automatically made or build. Makefiles are the traditional project equivalent offered by the commandline tools.
