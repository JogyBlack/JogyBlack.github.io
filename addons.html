Setup CS-RCS as BC5.0x Tool by David Arnold

CS-RCS is a freeware revision control system. This is how to be used from BC5.0x IDE:

Open the Borland IDE and goto "\Options\Tools" and choose "New", and enter the following information:

  • Name: CS-RCS Checkin
  • Path: C:\PROGRA~1\COMPON~1\CS-RCS\System\csrcs.exe (you need to use the short path to the actual location)
  • Command Line: $SAVE EDIT $NOSWAP $CAP MSG(generic) CheckIn "$EDNAME"
  • Menu Text: CS-RCS Checkin
  • Help Hint: Checks In Current Document to CS-RCS
Advanced Options:
  • Tool Type: Viewer
  • Place on Speed Menu
  • Applies To: .cpp;.hpp;.c;.h;.rc;.rh
