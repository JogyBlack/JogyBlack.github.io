Bulgarian Language Morphology




In 1996, together with Kiril Simov and Atanas Kiryakov, I began working on a Bulgarian language morphology project. Recently this project became part of OntoText Lab.

The software I created consists of two main programs - Slovnik - Windows GUI application which can be used to browse the full dictionary, to search for words, and to perform various analyzes upon text files.

The other is a server program which receiver words via TCP/IP sockets, and returns all forms of the given words. This server works under both Windows and Linux.
